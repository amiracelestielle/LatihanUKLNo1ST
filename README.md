# Program menghitung Rata-Rata Nilai Ujian

Program ini dibuat untuk membantu melakukan rekapitulasi nilai dari sejumlah siswa.  
Pengguna akan memasukkan jumlah siswa, lalu nilai ujian tiap siswa.  
Program akan menjumlahkan semua nilai dan menghitung rata-rata nilai ujian.
